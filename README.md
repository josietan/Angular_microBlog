# 微博项目
## 技术栈
* 界面：BootStrap + jQuery;
* 前端框架：Angular + Require.js + AngularUIRouter; <br>
* 后台：Node.js + ExpressJS; <br>
* 数据库：MongoDB + Mongoose; <br>
* Socketio
## 功能
* 注册会员
* 验证登录
* 发表微博
* 桌面提醒
* 微博评论
* 微博评论点评、点赞
* 删除别人的评论
* 删除自己的微博
* 个人主页能看见时间线
* 个人一句话签名、站内信、头像系统(支持头像剪裁)

## 此项目使用的是SPA;
## 路由是RESTful风格。

## 项目起步
* `cnpm install`
* `mongod --dbpath c:database`
* `node app`
